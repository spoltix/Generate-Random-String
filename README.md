Generate-Random-String
======================

Extension jQuery pour générer une chaine aléatoire


<code>$("input[name=pass]").randomString ();</code>
<code>
  $("input[name=pass]").randomString ({
      length:'8'
  });
</code>
<code>
  $("input[name=pass]").randomString ({
      length:'8',
      chaine:'ABCDEFGHYJKLMNOPQRSTUVWXYZ0123456789'
  });
</code>
